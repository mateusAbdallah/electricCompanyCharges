# Electric Company Charges

- An electric company is charging its consumers based on the following table:

| Numbers of Units Consumed | Charges/Unit |
|-------------------------- | ------------ |
|      :-----:                  :-----:
| For the first 200 units   | 2.5          |
| For the next 300 units    | 3.5          |
| Over 500 units            | 5            |
